# RTU Python CSP

## Kursa apraksts

Python datu apstrādes kurss pieaugušiem profesionāļiem ar pieredzi statistikā, darbā ar Excel un pamata zināšanām SQL. Kurss fokusējas uz Python izmantošanu datu apstrādei, automatizācijai un reproducējamām darba plūsmām.

## Mērķauditorija

Kurss paredzēts strādājošiem profesionāļiem statistikas nozarē ar dažāda līmeņa Python priekšzināšanām.

## Kursa struktūra

### 1. diena — Pamati un uzstādīšana

Dalībnieki iepazīstas ar Python lomu mūsdienu datu darba plūsmās, uzstāda lokālo darba vidi, apgūst atšķirības starp `.py` failiem un Jupyter piezīmju blokiem, kā arī nostiprina Python pamatus: mainīgos, tipus, `if`, `for`, sarakstus un virknes.

### 2. diena — Teksta apstrāde un tīrīšana

Uzmanība tiek veltīta neapstrādātu datu nolasīšanai, virkņu apstrādei, kodējuma problēmām, pamata `regex` lietojumam un datu tīrīšanas plūsmu veidošanai.

### 3. diena — Pandas pamati

Dalībnieki apgūst darbu ar tabulveida datiem, `DataFrame` struktūru, filtrēšanu, atlasi, grupēšanu, agregāciju, kārtošanu un datu kopu apvienošanu, pakāpeniski aizstājot tipiskas Excel darbības ar Pandas pieeju.

### 4. diena — Datu avoti un vizualizācija

Šajā daļā tiek savienoti vairāki datu avoti, nolasīti Excel, CSV un JSON faili, apskatīta pamata SQL integrācija, paplašināts Pandas pielietojums un diagrammu veidošana ar `matplotlib`.

### 5. diena — Mašīnmācīšanās pamati

Noslēdzošajā dienā tiek iepazīts mašīnmācīšanās darba process, apskatīta regresija, klasifikācija, datu sadalīšana `train/test` kopās un modeļu novērtēšana, izmantojot vienkāršus piemērus.

## Noslēguma projekts - neobligāts

Kursa beigās dalībnieki izveido pilnu datu konveijerapstŗādi, sākot no datu ievades un tīrīšanas līdz analīzei un vizualizācijai, demonstrējot apgūtās prasmes un reproducējamību. Projekts var ietvert arī vienkāršu mašīnmācīšanās modeli, ja laiks atļauj.:

```text
Ievade -> Tīrīšana -> Validācija -> Analīze -> Vizualizācija -> (Pēc izvēles: ML)
```

Projekta rezultātā tiek demonstrētas datu apstrādes prasmes, reproducējamība un analītiskā domāšana.
