# Python datu apstrādes kurss (5 dienas / 25 akadēmiskās stundas)

Strādājoši profesionāļi statistikas nozarē (Excel + nedaudz SQL, dažāda līmeņa Python pieredze)

# 1. diena — Pamati un uzstādīšana

## Mērķi
- Izprast Python lomu datu darba plūsmās
- Uzstādīt lokālo vidi
- Palaist Python skriptus un piezīmju blokus
- Apgūt pamata sintaksi

## Tēmas
- Python mūsdienu darba plūsmās (ETL, automatizācija)
- Lokāli pret mākoņi (VS Code pret Colab)
- Vides uzstādīšana (Python, VS Code, .venv)
- Pirmais skripts (hello_world.py)
- Jupyter piezīmju bloki pret .py skriptiem
- Python pamati:
  - Mainīgie, tipi
  - if, for
  - saraksti, virknes

## Uzdevumi
- Vienkārši aprēķini
- Vērtību filtrēšana
- Sarakstu pārveidošana

---

# 2. diena — Teksta apstrāde un tīrīšana

## Mērķi
- Nolasīt un attīrīt neapstrādātus datus
- Apstrādāt nekārtīgus ievades formātus

## Tēmas

- Kodējuma problēmas
- Virkņu apstrāde
- Pamata regex
- Tīrīšanas cauruļvadi

## Uzdevumi
- Attīrīt nekārtīgu datu kopu
- Standartizēt formātus
- Noņemt nederīgās rindas
- Eksportēt attīrītos datus
