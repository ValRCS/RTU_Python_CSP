# Python Data Processing Course (5 Days / 25 Academic Hours)

## Target Audience
Working professionals in a statistics department (Excel + some SQL, mixed Python experience)

## Course Theme
Python as a data processing, automation, and reproducibility tool for statistical workflows.

---

# Day 1 — Foundations & Setup

## Objectives
- Understand Python’s role in data workflows
- Set up local environment
- Run Python scripts and notebooks
- Learn core syntax

## Topics
- Python in modern workflows (ETL, automation)
- Local vs cloud (VS Code vs Colab)
- Environment setup (Python, VS Code, .venv)
- First script (hello_world.py)
- Jupyter notebooks vs .py scripts
- Python basics:
  - Variables, types
  - if, for
  - lists, strings

## Exercises
- Simple calculations
- Filtering values
- Transforming lists

---

# Day 2 — Text Processing & Cleaning

## Objectives
- Read and clean raw data
- Handle messy input formats

## Topics
- File reading (txt, csv)
- Encoding issues
- String processing
- Basic regex
- Cleaning pipelines

## Exercises
- Clean messy dataset
- Standardize formats
- Remove invalid rows
- Export cleaned data

---

# Day 3 — Pandas Fundamentals

## Objectives
- Work with tabular data
- Replace Excel workflows

## Topics
- DataFrame structure
- Filtering and selection
- Grouping and aggregation
- Sorting
- Merging datasets

## Exercises
- Recreate Excel operations in Pandas
- Group and summarize data
- Join datasets

---

# Day 4 — Data Sources & Visualization

## Objectives
- Integrate multiple data sources
- Create visual outputs

## Topics
- Reading Excel, CSV, JSON
- SQL integration (basic)
- Advanced Pandas (apply, datetime)
- Visualization (matplotlib)
- Automation concepts

## Exercises
- Multi-source pipeline
- Generate charts
- Build reusable script

---

# Day 5 — Machine Learning Basics

## Objectives
- Understand ML workflow
- Apply simple models

## Topics
- Supervised learning overview
- Regression
- Classification
- Train/test split
- Model evaluation

## Exercises
- Build regression model
- Build classification model
- Apply model to dataset

---

# Final Project

## Task
Build a complete data pipeline:
- Input → Cleaning → Validation → Analysis → Visualization → (Optional ML)

## Outcome
Students demonstrate:
- Data processing skills
- Reproducibility
- Analytical thinking

---

# Teaching Approach
- 30–40% instruction
- 60–70% hands-on
- Discussion-based learning
